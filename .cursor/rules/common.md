## 本プロジェクトの目的

- React + Spring Boot を利用した Web アプリケーションのサンプルリポジトリです。
- documents フォルダの lecture フォルダには教材としての資料を配置します。
  - 本リポジトリの実装内容にとらわれず、Web App, Docker, CI/CD などについて講座を開く際に利用する資料を配置します。

## 本プロジェクトの要件・機能

架空の BtoC グルメサイトの管理用の社内アプリを想定しています。

### (1) ユーザー・予約履歴確認機能

- ユーザーの一覧を確認することができます。
- ユーザーの詳細ページでユーザー情報及び、ユーザーの予約履歴を確認出来ます。

### (2) 店舗機能

- 飲食店の登録情報を確認することが出来ます
- 飲食店登録、詳細、更新、削除ができます
